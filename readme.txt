QYKCMS 4.3.0（后台版本：4.1.4）
====4.2.9至4.3.0更新说明====
1、修正部分浏览器屏蔽幻灯片广告图片的问题
2、修正搜索标签将隐藏分类也列入搜索的问题
3、完善搜索标签，新增sel及width参数，示例 {:search="news,help",sel=true,width=500}
4、修正开启静态缓存，设置的时间有效期不准确的问题
5、新增专题功能，调用标签 {:special}，详见官方博客标签大全
6、文章赞一下功能增加点击限制，后台可关闭，后台支持手动设置各选项值
7、忧化数据库读取过程，减少数据库读取次数
8、新增禁用删除站点设置、站点访问状态设置
9、修正部分PHP环境无法安装或无法进行环境检测的问题
10、修正开启伪静态时，评论或留言无法翻页的问题
11、修正因域名中包含“-”符号导致无法上传图片的问题
12、新增批量导入功能，支持直接指定本机文件夹
13、修正上一篇、下一篇文章的排序方式
14、修正执行系统升级后不重新登录会定时自动跳转到欢迎界面的问题
15、新增执行系统升级后显示更新日志
16、后台域名绑定备案号调整为非必填项，支持绑定IP
